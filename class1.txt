->id():
======
It is a predefine function in python.The main objective of id() function is to display the 
address of a variable or an identiifeir where our value is stored at memory allocation.


Ex1:
===
import time 
age=21 
print(age)
print()
print(id(age))
print()
time.sleep(2)
print("End of an application")
